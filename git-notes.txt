Git Config

git config --global user.email aaarabhi@gmail.com
git config --global credential.helper store (if typing username, password - this will store for longer)
 

Local Git 

git init (.git - hidden)	Initialize git in a new project
.gitignore (ignores files)	Copy .gitignore of respective program language - https://github.com/github/gitignore/blob/main/Node.gitignore
git add . (Can use vscode also)	Adds all files in current dir to git
git commit -m “<message>” (Can use vscode also)	Commits files
